## Unity version

- 2022.3.60f1

## Opening the project

1. Install Unity 2022.3.60f1 or a compatible Unity 2022.3 LTS editor.
2. Open this repository folder with Unity Hub.
3. Let Unity restore packages from `Packages/manifest.json`.

## Notes

Large generated folders such as `Library`, `Build`, `Build_F`, `Logs`, `obj`, and `UserSettings` are intentionally ignored.
Third-party assets may have their own license terms.
